# Projeto Escolar - Restaurante Sabor & Alma

Este repositorio contem um plano de trabalho completo para o trabalho escolar da empresa ficticia Restaurante Sabor & Alma.

O grupo desenvolve:

- Definicao da empresa ficticia
- Rede estruturada para restaurante
- Simulacao no Cisco Packet Tracer
- Orcamento tecnico
- Identidade visual e marketing

## Como usar estes documentos

1. Comecar pela definicao da empresa ficticia.
2. Passar ao planeamento geral.
3. Executar as fases de rede.
4. Montar simulacao e testes.
5. Fechar orcamento.
6. Produzir identidade visual e materiais de marketing.
7. Consolidar relatorio e apresentacao final.

No final, o grupo tera:
